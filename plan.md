# 討論SNS 開発計画書

## 1. 概要
ユーザー同士が1対1で討論を行うSNSを開発する。  
テーマは自由で、カテゴリごとに分類可能とする。  
匿名性を保ちながら公平な議論環境を提供する。

## 2. 開発目的
- 小規模な個人開発からスタート
- 最終的には公開サービスとして運用
- 議論・意見交換に特化したSNSの構築

## 4. 機能要件

### 4.1 ユーザー機能
- アカウント登録 / ログイン
- 匿名表示（ディベート単位）
- 通報機能
- ブロック機能

### 4.2 ディベート機能

#### ■ 流れ
1. ユーザーがテーマを作成
2. 他ユーザーが参加
3. 即時ディベート開始
4. 自由投稿で議論
5. 第三者が閲覧・投票

#### ■ 形式
- 1対1のディベート
- 自由投稿型（ターン制なし）
- 投稿は時系列表示

#### ■ 匿名仕様
- ディベートごとに固定
  - 参加者1 → A
  - 参加者2 → B

### 4.3 投稿機能
- テキストのみ
- 最大1500文字
- 編集・削除不可（ユーザー）
- 管理者のみ削除可能

### 4.4 投票機能
- 観戦者のみ投票可能
- 1ユーザー1票
- AまたはBに投票
- 参加者は投票不可

### 4.5 勝敗判定

#### ■ 終了条件
- 一定時間経過後に終了（例：24時間）

#### ■ 判定ルール
- 得票数が多い側が勝利
- 総投票数が一定未満の場合は無効試合（引き分け）
- 同票の場合も引き分け

### 4.6 通報機能

#### ■ 仕様
- 通報数5件で投稿を非表示
- 同一ユーザーの通報は1件扱い
- ディベートごとにリセット
- 通報理由は選択式

#### ■ 表示
- 通報された投稿は完全非表示

### 4.7 投稿制限
- 連投制限（例：15秒）
- （任意）投稿数上限

- created_at
- is_hidden

### votes
- id

- user_id

### reports
- id

- target_post_id
- reporter_id
- reason

### blocks
- id
- blocker_id
- blocked_id

## 6. API設計

### 認証
- POST /register
- POST /login

### ディベート
- POST /debates（作成）
- GET /debates（一覧）
- POST /debates/:id/join（参加）
- GET /debates/:id（詳細）

### 投稿
- POST /debates/:id/posts
- GET /debates/:id/posts

### 投票
- POST /debates/:id/vote

### 通報
- POST /reports
